

关于位流图的使用笔记:
    在使用位流图的过程中, 发现, 在使用软件生成dta文件的时候, 需要将格式选择成
    TRUE color with ahpha channel
    生成其他文件的话读取是存在问题的

关于驱动器细分设置:
    将驱动器设置为sw3细分,看注释可以知道转一圈需要6400个脉冲,程序中定时器3的分频
    系数为TIM_PWM_Init(500-1,84-1);
    在这顺便说一下如何计算分频系数,系统时钟为84,那么可以算出84000000/500/84=2000
    即每秒钟2000次,所以在这里得出电机转动一圈所需要的时间为6400/2000=3.2秒
    这样就可以实现不同电机的转速转动

关于伸缩距离控制:
    1.计算电机旋转一圈伸缩距离,由此得到到目标距离一共需要多少圈

    2.利用光耦传感返回的电压值,判断实时伸缩的距离,实时采集电压,直到电压值达到与之
        相对应的距离

系统正常运行编码器设置:
    将驱动器设置为800细分,即1 off(下)2\3\4 on(上),看注释可以知道转一圈需要800个脉冲,
    最新的程序使用的是定时器8,定时器的分频系数为实时设置,由于传动器转一圈向下走2mm,所
    以设置运动单位为400,即每次运动都是以1mm为距离的.根据按压深度的不同,实时的设置转一
    圈的速度,按压深度越大,旋转速度越快,当前设置的旋转速度宏是1500,即每分钟110加减3的
    频率,修改旋转宏为800时,频率为每分钟60次.
